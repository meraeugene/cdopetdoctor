# CDO Pet Doctor Landing Page

A modern static landing page for CDO Pet Doctor, a veterinary clinic in Cagayan de Oro. The page uses the clinic's green and yellow branding, local clinic imagery, responsive layouts, Lucide icons, Tailwind CSS, Google Fonts, and AOS scroll animations.

## Project Structure

```text
.
+-- index.html
+-- README.md
`-- assets/
    +-- cat.webp
    +-- cat2.webp
    +-- hero.webp
    +-- location.webp
    +-- service1.webp
    `-- service2.webp
```

## How To View

Open `index.html` directly in a browser.

The page depends on CDN links for Tailwind CSS, Google Fonts, Lucide icons, and AOS animations, so an internet connection is recommended when viewing it.
